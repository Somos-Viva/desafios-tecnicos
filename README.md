# Desafios de Código para candidatos a Desenvolvedor na Viva

> **Uso de IA generativa é permitido.** Você pode (e deve) usar ferramentas como ChatGPT/Copilot para acelerar.  
> **Mas você precisa entender o que está fazendo.** 

---

## Passo 0 - Fork este repositório

As suas respostas ao desafio devem estar dentro de um fork deste repositório. Sua resposta será o link para o seu fork.

Você pode Googlear como fazer isso.

## Desafio 1 - Flask e FastHTTP

Crie uma pasta intitulada "desafio_1" dentro do seu fork.

Nesta pasta, crie dois projetos:

Projeto Flask: Um servidor Flask que tem apenas uma página e endpoint, dizendo "Hello, World". Use a rota "GET /".

Projeto FastHTTP: Um servidor FastHTTP que tem apenas uma página e endpoint, dizendo "Hello, World". Use a rota "GET /".

Crie um README.md dentro da pasta "desafio_1" contendo instruções de como rodar cada projeto. O que eu tenho de instalar para rodar os projetos e como instalo? São os mesmos comandos para rodar no Windows, Linux, Raspberry Pi, MacOS? Como eu verifico que os dois servidores estão funcionando? É possível rodar os dois projetos ao mesmo tempo?

Crie uma seção ao fim do README.md explicando as diferenças entre os dois projetos, linha a linha. O ideal é usar uma tabela em Markdown.

## Desafio 2 - Produzindo código Python

Crie uma nova pasta para este desafio, com um novo README.md. Copie seu servidor Flask para esta nova pasta.

Para este desafio ***não queremos código, queremos prompts***: Como você faria uma IA generativa resolver isso para você, quais as dificuldades, qual o processo?

Suponha que o negócio tem algumas novas necessidades:

 - Uma nova página para exibir o conteúdo da planilha "editorias.csv" que está contida neste repositório.
 - Uma página que acessa [https://catfact.ninja/fact](https://catfact.ninja/fact) e exibe a resposta formatada.
 - Uma página em HTML5 que lê o horário do computador de quem está navegando no site e exibe o horário. Quando o mouse passa por cima do horário, a hora muda de cor.
 - Uma nova página que ligue a câmera do usuário e tire uma foto do usuários quando ele clicar em um botão.

Que prompts podem ser usados para gerar este código?

## Desafio 3 - Planejando um projeto

Precisamos programar um media player. Vamos carregar no media player uma playlist de arquivos mp4 e png, que serão exibidos sequencialmente na segunda tela do PC. O usuário deve poder, a qualquer momento, modificar por quanto tempo cada png é exibido e em qual ordem os arquivos são exibidos.
